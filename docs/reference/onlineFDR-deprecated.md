# Deprecated functions in package ‘onlineFDR’

These functions are provided for compatibility with older versions of
‘onlineFDR’ only, and will be defunct at the next release.

## Details

The following functions are deprecated and will be made defunct; use the
replacement indicated below:

- LORDdep:
  [`LORD`](https://dsrobertson.github.io/onlineFDR/reference/LORD.md)
  with version='dep'

- bonfInfinite:
  [`Alpha_spending`](https://dsrobertson.github.io/onlineFDR/reference/Alpha_spending.md)
